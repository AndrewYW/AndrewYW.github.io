Background

Ascii Chess was a project completed for my Software Methodology class.The program implements a 2 player Chess game, drawing the board and taking inputs from the Eclipse terminal. The program correctly identifies all chess moves, including recognizing checks, checkmates, illegal moves, as well as stalemates. Additionally, the project required that we create a UML diagram for the object-oriented design, as well as properly adding Javadoc tags and files.
