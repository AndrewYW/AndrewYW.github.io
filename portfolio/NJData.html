Background

NJData was a project completed over the course of a class I took, Principles of Information and Data Management. The class covered database concepts, focusing on relational database concepts using SQL and MySQL. NJData focuses on New Jersey's presidential election voting history from 2004 to 2016, covering a variety of factors including gender, GDP, population growth. The end result is a small web application hosted on a DigitalOcean Droplet VPS, using the Python Flask framework, running the SQL queries on a MySQL server.
